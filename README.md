# WhatsApp Clone (whatsapp-clone)
## App Setup

Clone the repository
```
git clone https://github.com/John-Weeks-Dev/whatsapp-clone.git
```

Now run this command to start the project 
```
yarn install

yarn serve
```

And to start the backend
```
cd backend

yarn install

yarn start
```

You should be good to go!
